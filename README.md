# RocoBoxQuickCheck - 洛克王国随从极速筛选与多开统计盒子

RocoBoxQuickCheck 是一款专为**洛克王国多开/快速刷宠玩家**量身定制的单页超高密度一阶随从极速对齐、筛选与计数统计工具。

它致力于在极其有限的屏幕空间（单页流式栅格）内，提供无需滚屏、瞬间直达的宠物秘语检索，并结合精准的**自动计数与动态占比统计系统**，完美契合多开窗口比对与产出效率记录的需求。

---

## 🌟 核心特色

### 1. 🔍 双速极速筛选引擎
*   **拼音/简拼定位**：支持首字母拼音简写（例如输入 `ydh` 即可瞬间秒定位到“银蝶后”，输入 `xgdd` 定位“炫光迪迪”）。
*   **关键词/密语模糊匹配**：支持输入游戏提示内包含的特殊描述性词汇（如“菊花”、“阳光”、“公平”、“极速”等），系统将自动使用高亮标记（`<mark>`）将匹配的信息字词呈现在卡片中。
*   **无感按键重置**：随时按 `Backspace` 删减字词，网格将瞬时回滚显示。

### 2. 📊 智能计数与动态占比计算（NEW）
*   **网格独立计数**：每只宠物卡片都带有醒目的 `+` 和 `-` 增减计数器。
*   **动态占比计算**：当发生任何计数时，卡片底部的“计数”标签下会自动展现**该宠物数量在所有已记录随从总数中的百分比占比**（`占比 = (单卡计数 / 已记录总数) * 100%`），方便玩家直观监控每个宠物的分布率或爆率。
*   **页脚全局汇总仪表盘**：
    *   **记录种类**：精确展示已抓获种数占总随从库的比例（例如 `5 / 23`）。
    *   **收集进度**：自动测算解锁收集率。
    *   **总计数汇总**：页脚同步反映抓获的所有箱子随从的总累积数。
    *   **防误触一键清空**：支持一键清归零所有网格计数，设有“请再次点击以确认”的双重防误触拦截层。

### 3. 🎨 极致的视觉质感与细节调优
*   **专属高品质头像**：精心适配了前18只游戏核心宠物的定制精绘 JPG 头像，告别模糊默认网格。
*   **视觉重构与底色对齐**：通过微调 CSS，将最后 6 只冷门/特殊属性格子的底色优雅地统一替换为温润自然的古朴灰色调（高级 RGB 配色 `#f2eadd`），使得视觉在紧凑间距中依旧具有极致的呼吸感和统一感。
*   **专属动态属性指示**：拥有特定属性的精灵（例如水属性“果冻”、虫属性“星辰虫”、光属性“公平鸽”等）配备了呼吸微动闪烁（`animate-pulse`）的属性徽标与指示，对齐更简单。

### 4. 📱 完美的多开屏幕适配 (Responsive Bento-Grid)
*   采用桌面第一精密兼顾移动响应式设计，极高密度布局。
*   在超宽屏、分屏、甚至小窗等复杂比例下自动缩放头像与按钮大小。
*   点击整张卡片即可快捷触发默认 `+1` 计数，无需费力寻找微小按钮。

## 🛠️ 技术栈
*   **前端框架**：React 18 + TypeScript (严格类型安全)
*   **构建工具**：Vite (毫秒级响应，零预编译等待)
*   **样式方案**：Tailwind CSS (极致原子化轻量布局，配合温纯的中性色调与大地色纸张底色)
*   **图标库**：Lucide React

## 🚀 本地开发与启动说明

1. **环境准备**：确保您本地已安装过 [Node.js](https://nodejs.org/)。
2. **克隆仓库**：
   ```bash
   git clone https://github.com/redc07/RocoBoxQuickCheck.git
   cd RocoBoxQuickCheck
   ```
3. **安装依赖**：
   ```bash
   npm install
   ```
4. **启动本地开发服务器**：
   ```bash
   npm run dev
   ```
   启动后可在浏览器中直接访问控制台输出的本地服务地址（如 `http://localhost:3000`）。
5. **编译构建**：
   ```bash
   npm run build
   ```
   一键输出压缩打包后的全静态资源于 `/dist` 目录中。
