# 洛克王国世界S2 · 盒子查询记录工具

本工具用于洛克王国第二赛季（S2）盒子精灵谜题的快速筛选、匹配与出货数据统计。

## 🌟 核心功能
- **极速拼音匹配**：支持通过全拼、拼音首字母缩写快速定位精灵。
- **提示词模糊检索**：支持输入盒子提示词中的关键词或前缀进行检索和高亮显示。
- **精灵数据统计**：支持对各只精灵进行单独计数（支持快捷增减），实时计算、展示精灵产出的占比数据。
- **数据报表导出**：支持一键将本地统计结果导出为 Excel 格式报表。

## 🛠️ 技术栈
- React + TypeScript
- Tailwind CSS
- Lucide React

## 🚀 启动与编译
1. 安装依赖：`npm install`
2. 启动开发服务器：`npm run dev`
3. 生产环境编译：`npm run build`
